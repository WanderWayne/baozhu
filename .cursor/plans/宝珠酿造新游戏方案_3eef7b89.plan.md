# 宝珠酿造 · 合成宇宙 新方案（V2）

## 一、游戏定位与总体目标

- **场景**：门店排队 / 等饮品的 3–5 分钟碎片时间；支持手机 + iPad 竖屏单手操作。
- **玩家体验**：
- 上来 30 秒内就能完成一次合成、看到小动画 → **微爽感**。
- 每一小关 1–3 分钟，有明确“门后有什么”的期待 → **关卡期待感**。
- 多局累积，逐步“修复宝珠酿造术”、解锁真实饮品 → **品牌认知 + 长期成就感**。
- **核心一句话**：
- **“看得见门后光的酿造工坊 + Infinite Craft 式自由合成 + 宝珠奶酪真实配方”**。

---

## 二、核心玩法与三层循环

### 2.1 微循环（3–10 秒）：合成反馈 + 小惊喜

- **基础操作**：底部原料槽拖拽到上方工作台，自由摆放；两个（或多个）物品重叠时尝试合成。
- **反馈要求**：
- 成功合成：0.3 秒内出现“跳起 +闪光+小音效”，新物品生成。
- 失败合成：不给红叉，只给“风味还差一点”的软提示。
- **设计要点**（利用 Infinite Craft 优点）：
- 大量**合理 + 一点点意外**的结果：
- 合理：牛奶 + 菌种 → 雪域酸奶。
- 微意外：雪域酸奶 + 冷冻 → 雪酪；雪酪 + 发酵 → 酒酿风味冰激淋。
- 每 3–6 次操作就给一次“新词 / 新原料 / 新小故事” → 持续小成就。

### 2.2 星盘循环（每关 1–3 分钟）：主题关卡 + 诱饵锁门

- 每一关 = 一个 **小主题工坊**：
- 例如：`奶酪谷`、`花之山脉`、`果野平原`、`谷物峡谷`、`温差雪峰`。
- 每关只引入 **1 个新机制**：
- 如“花香扩散”、“酸甜属性冲突”、“吸水膨胀”、“冷热变化”等。
- **必备结构**（从《关卡结构.md》第一关抽象出来）：

1. 开局就看见一扇 **半透明的酿造室门（诱饵锁）**，门后是本关“招牌饮品”轮廓。
2. 玩家在开放工作台里自由尝试合成（Infinite Craft 感）。
3. 每达成 1–2 个关键合成，门的状态升级一次：微光 → 震动 → 光纹汇聚 → 开门。
4. 开门时给一次“仪式感极强”的动画 + 品牌故事句。

- 这样得到：
- **Infinite Craft 的自由尝试 + Zelda 式可视化目标门 + It Takes Two 式“一关一主题机制”**。

### 2.3 宏循环（多局）：重建“宝珠酿造术”的世界

- **主线设定**（沿用 `~~~.md` 的非关卡部分）：
- 一场“时间失序”让宝珠的经典配方全部消失，
- 玩家作为新晋酿造学徒，要通过合成找回：
- 酪系列 → 奶香之谷
- 酸奶昔系列 → 冰雪山脉
- 热饮系列 → 炉火厅堂
- 酸奶碗系列 → 谷物峡谷
- 冰奶系列 & 单品 → 雪域源泉
- **宏观目标**：
- 找回每个系列中的 1–2 款代表性饮品（来自 `产品配料.md`），逐渐点亮地图节点。
- 终局目标：合成传说中的“**天赐宝珠酪**”（需要所有系列核心饮品齐全才可解锁）。

## 三、总体界面与结构设计

### 3.1 入口 & 开场动画（简化后的版本）

- **黑屏 + 白点**：
- 全黑背景中央，一个缓慢呼吸的白点，点击范围远大于视觉大小（方便手机 / iPad）。
- 文案（灰字）：“轻触，唤醒酿造之境。”
- **点击后**：
- 白点扩散成宝珠主题色圆圈 → 渐变为温暖的工坊背景。
- 中央出现一句：
- “十三年慢酿的秘法，被时间吹散。”
- “请你，帮我们把味道找回来。”
- 淡出后进入 **世界地图 / 关卡选择界面**。

### 3.2 世界地图 / 主界面

- 布局（竖屏）：
- 上方 30%：极简世界地图（像手绘菜单），几个点亮 / 灰色的小区域：奶酪谷、花之山脉、果野平原、谷物峡谷、温差雪峰、终极圣殿。
- 中间 50%：当前世界的“关卡门阵列”（1–3 个小门），每扇门上有对应饮品小插画轮廓（未解锁为剪影）。
- 下方 20%：
- 文本条：展示当前选中门的故事一句。
- 按钮：`开始关卡 / 再次挑战`。
- 视觉风格：大量留白、手绘线稿 + 少量写实食材照片或小插画，延续品牌“中式、极简、写实”调性。

### 3.3 关卡内界面

- 竖屏 3 区：

1. **上方 25%：可视化目标门区**

- 半透明圆拱门，后面是本关目标饮品的朦胧光影。
- 门旁有一行小字：
- “本关目标：酒酿桂花酪 / 三重莓果酸奶昔 ……”

2. **中间 55%：自由合成工作台**

- 类似木桌纹理的空区域，所有拖出来的元素都可以自由摆放。
- 状态效果（花香环、冰雾、气泡）在此层表现主题机制。

3. **底部 20%：原料带 / 工具栏**

- 水平滚动槽，放置当前关可用的基础原料 + 工具（如冷冻、加热、切碎等）。
- 交互要求：
- 单手大拇指即可拖拽、缩放不多、元素尺寸足够大。
- 合成结果、升级、失败提示尽量靠近操作位置，减少视线跳转。

## 四、关卡结构总览（世界 → 章节 → 代表饮品）

### 4.1 世界与系列映射（对应 `产品配料.md`）

1. **世界 1：奶酪谷（乳制基础）**

- 目标系列：单品系列 + 酪系列基础（冰酥酪、雪域酸奶、酒酿酸奶、无糖酸奶）。
- 代表关卡目标：
- 1-1：冰酥酪
- 1-2：雪域酸奶
- 1-3：酒酿桂花酪（简化版）

2. **世界 2：花之山脉（花香系统）**

- 对应产品：酒酿桂花酪、酒酿桂花酪（自然发酵甜）、酒酿玫瑰酪、岩间茉莉酸奶昔等。

3. **世界 3：果野平原（水果属性）**

- 对应产品：珍珠芭乐草莓酪、三重莓果 VC 酸奶昔、青青柚子酪等。

4. **世界 4：谷物峡谷（谷物与坚果）**

- 对应产品：0 蔗糖脆烤坚果酸奶碗、可可麦圈酸奶碗、坚果芒芒酸奶碗等。

5. **世界 5：温差雪峰（冷热与热饮）**

- 对应产品：听泉茉白、五鼎芝玫瑰烤奶、秋酿 桂花山药酒酿牛乳、活酒酿冰奶等。

6. **终极圣殿：宝珠大成**

- 目标：合成“天赐宝珠酪”（虚构终极饮品）
- 条件：前述系列中，每个系列至少完成 1–2 款核心饮品。

> 注：每关时长控制在 1–3 分钟；世界 1 的前两关更短（30–60 秒），用于友好 onboarding。

## 五、重点：第一世界 & 第一关（结合 Bob-omb + 诱饵锁 + Infinite Craft）

### 5.1 世界 1：奶酪谷结构

- 主题：从“牛奶”到“酪”的基础认知，建立酿造语言。
- 机制：
- 发酵（时间） → 牛奶 → 酸奶 → 酪。
- 冷冻（温度） → 酸奶 → 雪酪。
- 关卡：
- **1-1 冰酥门廊**：玩家学习合成冰酥酪，门后可见一小碗白色冰酪光影。
- **1-2 酸奶阶梯**：玩家合成雪域酸奶，门后是一只玻璃杯酸奶身影。
- **1-3 奶酪试炼**：玩家第一次做出“酒酿桂花酪 简化版”，门后第一次出现花影与酒酿光晕。

### 5.2 关卡 1-1：冰酥门廊（完整节奏）

**（1）开局 30 秒：探索 + 可视化目标**

- 画面：
- 前景是质朴木桌，上有 `牛奶`、`酒酿原浆`、`冰糖碎`、`冰块` 四个基础图标。
- 远处上方有一扇圆拱形半透明门，门后模糊一小碗发光的白色冰酪轮廓。
- 文案浮层：
- “把味道拼回去。先从最简单的一碗开始。”

**（2）友好实验：显而易见的合成（学习规则）**

- 系统给出 2–3 个“低悬果实”组合：
- 牛奶 + 冰糖碎 → 甜牛奶。
- 牛奶 + 冰块 → 冰牛奶。
- 冰牛奶 + 酒酿原浆 → 酒酿冰奶基础液。
- 反馈：
- 每次成功合成：小气泡、透明提示：“你让奶更接近宝珠的味道了。”

**（3）小挫折：失败不挫败**

- 若玩家乱配：酒酿原浆 + 冰块 / 酒酿原浆 + 酒酿原浆 等：
- 提示词：
- “这碗酒酿有点孤单，似乎想找点奶香。”

**（4）关键物品 1：冰酥基底 → 门第一次响应**

- 关键合成：
- 甜牛奶 + 酒酿原浆 → 酿香奶底。
- 酿香奶底 + 冰块 → 冰酥基底。
- 触发：
- 门边缘亮起一圈柔光。
- 文案：
- “酿造室似乎闻到了奶香。”

**（5）关键物品 2：冰酥酪 → 开门高潮**

- 最终合成：
- 冰酥基底 + 少量酒酿原浆 → **冰酥酪**（目标物）。
- 动画：
- 门锁图案从外围向中心依次点亮，轻微震动，门缓缓开启。
- 门后出现下一世界“奶酪谷·深处”的一角剪影（预告 1-2 关）。
- 完成提示：
- 大字：“你找回了宝珠的第一碗冰酥酪。”
- 小字：“牛奶、米酒和冰糖碎，最简单的组合，需要最长的耐心。”（品牌文案自 `Information.md` 提炼）

**（6）导出宏循环：**

- 结算界面：
- 展示：冰酥酪插画 + 小卡片：
- “现实中的冰酥酪：牛奶、米酒、冰糖碎，零添加的小甜甜。”
- 按钮：
- `继续 · 雪域酸奶`（进入 1-2）
- `返回工坊`（回到世界地图）

> 这一关综合了：Bob-omb 的友好教程、Zelda 的可视化门、Infinite Craft 的自由合成尝试。

## 六、后续世界关卡设计思路（简要）

这里只列出每世界代表关的核心机制与品牌挂钩点，细节可在实现阶段展开。

### 6.1 世界 2：花之山脉（花香扩散）

- 代表关卡目标：酒酿桂花酪、酒酿玫瑰酪、岩间茉莉酸奶昔。
- 额外机制：
- 花类原料放在桌面时，会在周围形成花环光晕：
- 桌面上的其他乳制品被“芳香化”，升级为花香款。
- 花粉过多时，会让某些饮品“失衡”（变苦 / 太腻），玩家要用水或冰块“稀释”。
- Zelda 式门：门后可见一串桂花雨，随着进度加深，桂花从门缝中飘出。

### 6.2 世界 3：果野平原（酸甜属性）

- 代表关卡目标：珍珠芭乐草莓酪、三重莓果 VC 酸奶昔、青青柚子酪。
- 额外机制：
- 水果有“酸 / 甜 / 清爽 / 浓郁”标签：
- 酸 + 乳 → 可能“凝结失败”或变成“重口味实验品”。
- 甜 + 乳 → 升级为香甜款。
- 玩家可以使用“打泥工具”将水果变为果泥，解锁新分支配方。
- 门预告：门后是色彩丰富的果树剪影，通关时果树开满对应水果。

### 6.3 世界 4：谷物峡谷（爆爆珠 & 坚果物理）

- 代表关卡目标：0 蔗糖脆烤坚果酸奶碗、可可麦圈酸奶碗、坚果芒芒酸奶碗。
- 额外机制：
- 奇亚籽：放进液体后，几秒后自动“膨胀升级”为口感更佳的版本。
- 爆爆珠：遇热会“爆开”生成新词，可能触发连锁反应。
- 坚果 & 谷物：可以烤制 / 碎裂，改变口感与名称（例如“脆烤坚果”）。

### 6.4 世界 5：温差雪峰（冷热平衡）

- 代表关卡目标：听泉茉白、五鼎芝玫瑰烤奶、秋酿 桂花山药酒酿牛乳、活酒酿冰奶。
- 额外机制：
- 每个词都有“温度状态”：冷 / 常温 / 热。
- 加热会增加香气（升级）、去冰；冷冻会生成雪酪或裂变成冰沙版本。
- 某些配方只有在“刚刚好”的温度窗口可合成（营造一点“调温”乐趣，但不复杂）。

## 七、提示词与文案设计（示例）

### 7.1 新手提示（第一世界）

- 初次拖拽时：
- “把两样放得更近一点，味道就会自己说话。”
- 第一次失败合成：
- “似乎还缺一点什么，不如换个搭配？”
- 玩家停顿 5 秒未操作：
- “门后的光在等你试一试新组合。”

### 7.2 成功 & 文化小知识（结合 `Information.md`）

- 合成出酒酿相关：
- “四十天的慢酿，让糯米在静默里变成酒香。”
- 合成出经典花饮：
- “桂花，是秋天最温柔的香气。”
- 通关奶酪相关世界：
- “从一碗酪饮到今天的菜单，宝珠走了十三年。”

### 7.3 门的三个阶段提示

1. 门首次亮起：

- “酿造室似乎被唤醒了。”

2. 门轻微震动：

- “好像只差最后一步味道。”

3. 门打开：

- “这一角酿造之境，被你重新点亮。”

## 八、故事线（软叙事）

- **序章：失序的配方**
- 文案：
- “时间在酒缸里走错了路，老配方散落成字句与原料。”
- “你拾起的每一粒米、每一朵花，都是找回味道的一小步。”
- **第一卷：奶香之谷** — 找回最基础的酪与酸奶。
- **第二卷：花之山脉** — 重新点亮桂花、玫瑰、茉莉的香路。
- **第三卷：果野平原** — 让莓果与柚子再次撞进雪酪里。
- **第四卷：谷物峡谷** — 让爆爆珠、奇亚籽和坚果重新发出声音。
- **第五卷：温差雪峰** — 在冷热之间找回“冬日一杯热饮”的慰藉。
- **终章：宝珠大成**：
- 当玩家合成所有系列的代表饮品后，解锁终极关：
- 文案：
- “十三年日落月升，终归于一碗。”
- “天赐宝珠酪，不是新配方，而是所有味道的和声。”

## 九、实现建议（为后续开发准备的高层指引）

- 技术上仍可沿用现有拖拽 + 合成 + 倒计时逻辑，只是：
- **数据结构上**明确区分：
- 基础原料 / 中间状态 / 终极饮品 / 世界与系列映射。
- **UI 上**重点增加：
- 顶部“可视化目标门”与 3 阶段状态。
- 世界地图 / 关卡门阵列页面。
- 游戏节奏：
- 前两关极短（30–60 秒），只讲操作 + 门机制。
- 之后关卡增加“主题机制”，但每关只加一层，不叠太多规则。
- 商业与品牌：
- 每个现实饮品通关卡片上展示 **真实产品小图 + 简短描述**，引导用户在店内“点来试试”。
- 可选：完成某个系列全部关卡后，给一张“酿造师纪念徽章”截图，用于线下出示给店员获得小福利。
