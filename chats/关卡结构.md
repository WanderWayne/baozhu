#🧭 **关卡结构 V1.0：

《宝珠酿造术》第一世界：宝珠奶酪初启之境**

地图结构：

这是一个半开放的小小“酿造工坊”

玩家能看到 一扇打不开的半透明酿造室大门（诱饵锁）

桌面上摆着几个小工具

四周散落未完成的小物件（玩家可以通过合成获得）

氛围温暖、带点国风 & 中式甜品店感觉

核心体验目标：
让玩家感觉：

“我正在一步步进入一个更厉害的酿造世界。”

#🔥 第一关结构（节奏曲线）
🎵 节奏模型：

探索 → 掌握 → 挫折 → 领悟 → 成长 → 奖励 → 新的期待

对应关卡事件如下：

##① 初始区域（30秒即可理解系统）—探索期

玩家一开始就能看到：

✨ 工作台上 3–4 个基础词语（如 “奶”、"茶"、“果”、“冰”）

✨ 一扇 半透明的酿造室门（能看见里面 faint 的光，但完全打不开）

门后若隐若现的东西：

桂花光影

或一杯正在发亮的“宝珠奶酪原型”

或神秘器具剪影

作用：
建立 Zelda 式“可视化目标（Visible Goal）”。

玩家此时会自然想：

“我要怎么进去？里面有什么？”

##② 玩家开始做第一次合成（兴趣期）

系统给 1~2 个“显而易见的组合”：

奶 + 茶 → 奶茶

奶 + 果 → 奶果

果 + 冰 → 冰果

茶 + 冰 → 冰茶

这些是“掌握感奖励（Mastery Reward）”
因为玩家首次成功合成 → 立即获得正反馈。

奖励设计：

每次成功合成一个基础物品：

小音效“噗—”像茶起泡泡

小动画：词语轻轻跳起，变成新物品

玩家心理奖励：
✔“我会玩了，我懂了”
✔“这游戏挺爽”

（此阶段不发真实奖励，如优惠券）

##③ 玩家会遇到“失败组合” （微挫折）

例如：

奶 + 奶 → ❌

冰 + 冰 → ❌

系统提示不是“失败”，而是：

“似乎还缺少某种风味…”

心理效果：

挫折控制在安全范围

引发探索欲

自己悟到组合后会更开心

（这里模仿 Zelda 神庙的小挫折—不强制，不惩罚）

##④ 玩家合成出“关键物品”后 → 激活诱饵锁门

当玩家合成出 「宝珠」（例如：糖 + 冰 → 宝珠）：

酿造室门出现微光

门纹开始亮一圈

文本：
“酿造室似乎被唤醒了…”

这是预告奖励（Foreshadowed Reward）
Zelda 神庙里最重要的心理设计。

玩家此时产生：

“哦，我越来越接近终点了！”

##⑤ 合成第二个关键物品 → 门震动（期待高潮）

例如：

奶 + 果/宝珠/茶 → 宝珠奶酪（朦胧版）

或找到“混合器”道具

门出现以下特效：

轻微震动

光纹开始向中心汇聚

门后的影子变得更清晰

文本：
“好像只差最后一步…”

这是 It Takes Two 风格的节奏升温。

##⑥ 最终合成成功 → 开门（高潮）

当玩家成功合成“真正的宝珠奶酪”的一次版本：

门闪光

锁纹从外到内亮起

门缓缓打开

光溢出

内部展示一个“主题区域的核心物品 / 工具”

例如：

✨ 宝珠萃取装置（新词生成器）
或
✨ 区域地图碎片（开启第二关）

物质奖励（第一关核心）：

解锁“新原料”用于下一关（如「桂花」「烤奶」「拉丝」）

解锁一个“更漂亮的酿造台皮肤”

心理奖励：

“我做对了”，“我进步了”

“下一关会更好玩？”

类似 Zelda 神庙那个“打开宝箱的爽点”

这是第一关的主要 dopamine 释放点。

#✨ 角色成长系统（物质 & 心理）
##1）物质成长（Playable Growth）

玩家获得：

🔓 新原料（增加组合树深度）

🔓 小装饰（桌布、器具、瓶子）

🔓 词语槽数量 +1（允许更长的合成链）

🔓 工坊等级提升

物质成长是“功能性奖励 + 收集性奖励”并存。

##2）玩家心理成长（Inner Growth）

第一关让玩家建立：

✔ 有“目标”的体验

看到门 → 打开门

✔ 有“我变强了”的体验

原本不会合成 → 会合成
原本打不开门 → 打开了

✔ 有“期待下一关”的体验

门后展示了下个主题的小预告
例如：

第二区域是“桂花酿”

或“藏在华容巷的秘密饮品术”

这种预告 = Zelda 神庙出门后看到大世界。

#🌀 循环系统（上瘾核心）

第一关就建立 4 条循环：

循环 1：可视化目标（门）循环

→ 看见
→ 接近
→ 未解锁
→ 尝试
→ 解锁
→ 新门出现

循环 2：合成实验循环

→ 新原料
→ 新组合
→ 新惊喜
→ 新物品驱动下一步

（类似 Infinite Craft）

循环 3：工坊成长循环

→ 工坊变漂亮
→ 看得见的提升（桌布、光效）
→ 动机增强

（类似“我的最强奶茶店”）

循环 4：剧情或氛围循环

→ 每关一点点揭开宝珠奶酪的秘密
→ 触发探索欲
