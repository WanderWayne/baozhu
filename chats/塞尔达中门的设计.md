在《塞尔达传说：旷野之息》与《王国之泪》的神庙里，你经常会看到一种 “能看到里面但开不了” 的门：

玻璃窗

栅栏

网格门

半透明能量壁

透过去能看到宝箱 / 终点 / 石像

但你现在还无法通过

➡ 这种门在关卡设计学里有一个非常明确的名字和作用。
我将讲解它属于哪一类设计、它的目的是什么，以及你如何把它迁移到你的宝珠酿造小游戏中。

#🌀 一、这种“看得见却进不去的门”属于哪种关卡设计？
🎖 **这类设计的专业名称叫：

“诱饵锁”（Foreshadowed Lock）
或
“可视化目标”（Visible Goal）
或
“前置期待”（Pre-Reveal Gate）**

游戏设计理论中的术语包括：

名称	含义
Foreshadowed Lock（提前预告的锁）	先让你看到奖励，再让你知道你还没资格打开
Teaser Gate（吊胃口的门）	明示“里面有好东西”，吊起动机
Visible Goal（可见目标）	在你继续努力前，让最终奖励一直在视线里
Readability Gate（可读性关卡结构）	用视觉告诉玩家“那是终点，但你需要方法”

Zelda 的神庙几乎每个谜题都在用这个手法。

#🎯 二、为什么 Zelda 神庙一定要用这种门？

这种门的目的非常明确：

1. 激发“我一定要进去”的动机（超强驱动力）

如果你看不到奖励，你会：

没方向

没动力

不知道是否值得继续研究谜题

但当你清楚地看见宝箱就在玻璃后面时：

→ “我一定要进去！”
→ 这立刻建立目标感

（这比给奖励本身更重要）

2. 帮助玩家理解：这关的目标是什么

这就是“可读性设计”。

Zelda 不用文字告诉你“终点在那边”，
而是用视觉语言告诉你：

“那个宝箱就是最终奖励”

“路径已经在那里，你只需要找到解法”

3. 帮助控制节奏（节奏管理）

这种门允许：

提前展示高潮（宝箱）

但延迟真正触发（打开的过程）

节奏变成：

看到奖励 → 产生期待 → 解谜 → 获得奖励 → 开心 → 下一间神庙

这是典型的 dopamine 节奏设计。

4. 奖励并非宝箱，而是“解开宝箱的过程”

Zelda 的神庙奖励是：

做到了 / 我解开了一个巧妙的机关

不是宝箱里那点装备。

同样地，你的游戏奖励也不应该全靠优惠券。

#🌈 三、你可以 1:1 把这种“诱饵锁”放入你的宝珠酿造小游戏里
🎯 用到你的游戏第一关：

“能看见但暂时不能打开的门”
将是玩家最强的好奇动机来源。

我帮你转化成你的游戏版本👇

🥛 A. 你的“诱饵锁门”可以这样表现：
状态：

第一关一开始，背景就有一个“半透明的酿造室门”：

里面亮着柔光

隐约能看到“桂花飘动的影子”

或一个未完成的饮品轮廓

玩家随时都能看到它

但无法互动

就像 Zelda 神庙一样：
你看到终点，但你还去不了。

🎁 B. 玩家完成 70% 的第一关后：

屏幕出现：

一丝光晕从门缝漏出来

短暂闪光

但仍然打不开

类似 Zelda 里“你知道快解开了，但还差一步”。

这个阶段就是：

foreshadowed reward（预告奖励）

🚪 C. 玩家完成关键合成后：

你触发“大奖励动画”：

门终于亮起

灯纹从锁的周围开始顺时针亮

轻微震动

门 slowly 打开

背后出现下一关主题色

文本：
「酿造之境的一角被唤醒了。」

这一瞬间就是：

等待长时间后终于打开的 Zelda 神庙宝箱门

玩家会深刻记住这一刻。

#🌟 四、为什么你的游戏第一关必须有这种“看得到但开不了的门”？
**因为它能补上 Infinite Craft 本身没有的东西：

关卡期待感。**

Infinite Craft 没有“终点” → 所以长线成就感弱。

你加入“看得见的门”之后：

✔ 玩家在玩的时候清楚“我在朝那扇门努力”
✔ 有明确的“愿望对象”
✔ 有 Zelda 式的神秘感
✔ 有 It Takes Two 那种“期待下一幕”的节奏
✔ 不需要发优惠券，也能产生成就感
