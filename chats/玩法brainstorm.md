### 玩法 Brainstorm：让“可交互的不只是物品”（区域/机关/地图）成为解谜核心

> 目标：把 Infinite Craft 的“高反馈合成探索”，装进更像解谜游戏/MV 的“空间规则 + 机关因果链”里。  
> 手段：增加**场景动词**（可交互区域/机关），并把“能否合成/合成结果”与“地点/状态”绑定。

---

### 1) 可交互区域（Stations / Zones）：某些合成必须在特定地点发生

- **酿造工坊台（基础）**：普通合成只能在工作台区域发生；离开工作台就无法“叠合”。
- **冰室 / 冷却台（❄️）**：任何物品进入该区会获得 `cold` 标签；冷相关配方只在这里可触发（如“冻”“冰镇”“凝固”）。
- **炉火 / 熬煮台（🔥）**：进入获得 `hot` 标签；支持“熬”“烤”“融化”“焦糖化”；同时有“烧毁风险”（超时变“焦糊”）。
- **发酵缸 / 静置区（🦠）**：只有放进缸里才能开始计时合成（你已有 timer 合成，可把“开始计时”改为必须在缸内触发）。
- **花窖 / 置香区（🌸）**：花类合成限定区域；或作为“香气污染/叠香”系统（在花窖放得越久香型越强/越混杂）。
- **研磨台 / 石臼（🪨）**：支持“捣碎/碾磨/粉化”动词；把“提取”从长按按钮，转为“拖到研磨台才能提取”。
- **过滤台（🧺）**：液体相关的“过滤/澄清”；能把“浑浊”变“清澈”，影响门的审判。
- **兑水泉 / 水槽（💧）**：可给物品添加 `wet`/`dilute`；也可作为“取水”的来源（解决按钮着火一类机关）。
- **盐糖料台（🧂🍬）**：调味站，能让玩家“精调属性”（甜度/咸度/酸度），把 Infinite Craft 的无目标扩张变成“达标解谜”。

---

### 2) 地图槽位 / 摆放型通关：把“合成结果”放进背景凹槽

- **凹槽门阵（🕳️🕳️🕳️）**：关卡不是献上一个目标，而是要把 2–4 个关键物品分别放进凹槽（顺序/位置正确才通关）。
- **形状/类型匹配槽**：槽位只接受 `type-tool` / `type-liquid` / `type-floral` 之类；或接受“热/冷”状态匹配。
- **“缺失配方图谱”拼图**：凹槽代表“工艺链的步骤”（冷却/发酵/过滤/置香…），每个槽位要放对应“阶段产物”。
- **称重托盘（⚖️）**：两个托盘平衡才开门（需要用“糖/水/果”调配重量或数量）。
- **三盏灯/三味审判（💡💡💡）**：槽位不是物品，而是“香/甜/口感”三项达标时点亮（物品放上去会显示属性条）。

---

### 3) 场景机关模板（因果链）：把“合成/提取”变成解决环境问题的工具

- **笼子 + 着火按钮（你给的例子）**：按钮被火焰封住 → 需要“水/冰/湿布”灭火 → 开笼子拿关键原料。
- **水闸/水位机关（💧）**：水位高低决定某块区域能否进入；合成“阀门/塞子/浮子”控制水位。
- **冰封通道（🧊）**：门口结冰，热区合成“热饮/热石”融冰；或冷区冻结水形成桥。
- **风扇/风道（🌬️）**：把轻物吹走/吹到槽里；合成“重物/粘性”抵抗；或用“湿”让物品变重。
- **黏手糖浆地（🍯）**：物品进入会慢/粘住；需要“油/粉/冰”解除；用于制造路径规划与节奏。
- **变色灯/滤光片（🔴🟢🔵）**：不同颜色照到物品，会改变其可合成性（只在绿光下发酵、红光下熬煮…）。
- **碎裂地板（🧩）**：需要用“胶/糖浆/奶酪”修补才能放置重物；把“材料的物性”变成解谜工具。
- **锁孔门（🔒）**：不是钥匙，而是“口味钥匙”：需要带某标签（桂花香/冷/发酵/清澈）才能插入。

---

### 4) “空间动词”更像 MV：背景区块可旋转/切换，改变可达性与规则

- **旋转台面模块（⤾）**：工作台分成几块可旋转的区域，旋转会改变哪些站点相邻（只有相邻才能合成/传递状态）。
- **桥梁翻转（⇄）**：翻转后“热区/冷区”互换位置，要求玩家规划顺序（先做热处理再翻去冷却）。
- **视角/层级切换（前景/背景）**：同一物品在不同层级上“可/不可合成”；靠切换让路径成立（MV 的错觉感用 UI 层级模拟）。
- **开关改变规则而不是开门**：例如开关让“水槽变热汤池”、让“花窖变干燥室”，玩家理解规则后会有“啊哈”。

---

### 5) 与“宝珠文化”强绑定的机关与素材（让世界观变成玩法）

- **店内真实工艺链**：浸泡→冷却→发酵→过滤→置香→装杯，每一步对应一个 Station。
- **“配方传承”机制**：某些关键配方不是靠试出来，而是通过机关救出“老配方纸/师傅口述”获得“提示/标签解锁”。
- **“十三年记忆”场景物**：田子坊灯、店门铃、旧菜单、玻璃罐墙——它们都能做成互动点：触发碎片、改变站点状态、解锁区域。

---

### 6) 让 Infinite Craft 不无聊：把“无限产出”变成“有目标的约束解谜”

- **订单约束（关卡条件）**：
  - **温度约束**：必须是冷/热，或“冷热对比”（先热后冷的“回甘”）。
  - **香型约束**：桂花/玫瑰/茉莉必须其一，或“不能混香”。
  - **口感约束**：脆/糯/滑/爆；用研磨/冷却/过滤等动词影响它。
  - **工艺约束**：必须包含“发酵/浸泡/熬煮”等步骤（由 station 记录）。
  - **步数约束**：4 步以内完成（把穷举变成优化题）。
- **“产出分层”**：
  - **普通产出**：规则生成（保证不空转）。
  - **关键产出**：手工配方（绑定碎片/成就/门变化）。
  - **传说产出**：强条件（夜晚/连续成功/特定站点顺序）+ 强仪式反馈（画面/音色）。

### 7) 关卡模板库（可以快速做很多“好玩但不贵”的关）

- **模板A：救援关键原料**：笼子/封条/结冰/水位 → 解机关 → 拿到关键原料 → 合成目标 → 献上/凹槽通关。
- **模板B：站点顺序谜题**：同样两种原料，不同站点顺序得到不同结果（先热后冷 vs 先冷后热）。
- **模板C：属性审判门**：门显示 3 条属性条，玩家需要调到范围内（不告诉配方，只告诉“差一点点”）。
- **模板D：搬运与污染**：穿过花窖会沾香，穿过炉火会变热；路径选择决定成败（空间=谜题）。
- **模板E：多槽位拼装**：做出 3 个部件分别放入凹槽，组合成最终“仪式之物”，开门进入下一幕。

### 8) 快速可视化建议（让玩家“看见世界在动”，更不无聊）

- **站点高亮**：物品进入某区域，区域泛光，提示“这里发生了变化”（冷雾/热浪/花瓣/气泡）。
- **状态贴纸**：物品角落显示小 icon（❄️🔥🦠💧🌸）表示当前状态/经历过的工艺。
- **机关可读性**：锁/火/水位/笼子都有“下一步缺什么”的可视线索（例如火焰旁有“缺水”的符号）。

### 9) 从《沉浸感加成建议》筛出的“低副作用沉浸机制”（建议默认开）

> 原则：不增加规则负担、不打断节奏、几乎不引入失败惩罚；主要靠节奏/反馈/“世界回应你”提升沉浸。

- **揭晓延迟（非赌博）**：合成先显示 `???（酿造中）` + 轻微旋转/冒泡，1–2 秒后揭晓（你已做一部分“神秘揭晓”，可以扩大到更多关键物）。
  - 注意：延迟要短、可跳过（玩家再次拖动/长按可立即揭晓），避免拖沓。
- **信息只给 70%（用暗示替代解释）**：失败不弹硬文字教程，改为环境暗示（门共振/光冷/低频音），让玩家“感觉还差点什么”。
- **世界反馈系统（失败也回应）**：
  - **正确方向**：光变暖、音乐上扬、门轻共振
  - **接近正确**：门更明显呼吸/闪烁
  - **完全错误**：环境变冷、音色变薄（但不惩罚）
- **仪式感合成**：合成不是“瞬间换物”，而是“旋转→对齐→靠近→揭晓”的小仪式（尤其是首次发现/关键配方）。
- **全场屏息 0.3 秒**：合成完成或门开启前瞬间“轻静音/降噪”再回弹，强化“重要时刻”。
- **门的呼吸与记忆**：门不是静态图，门光纹层层点亮并“记住你做到哪一步”（你已有 doorStage，可加更连续的“温暖度”/“纹理层”）。
- **连续成功让世界变温暖**：连成 2–3 次，背景色温/粒子/环境音逐步变暖；失败则慢慢回落（不清零，避免打击）。
- **安静/空置时的环境变化**：工作台空置一段时间，世界慢慢暗下来；玩家再次拖入物品，世界被“唤醒”。
- **重要物品命名慢慢写出**：关键产物的名字逐字显现（而非瞬间出现），增加“被认可为真实存在”的感觉。

### 10) 从《玩法与动作系统设计》筛出的“低副作用动作/解谜规则”（建议逐关引入）

> 原则：动词少但可组合；每关只教一个新动作（teach → remix → twist）。

#### 10.1 第一层（安全动作，几乎零副作用）

- **靠近（Approach）**：某些效果只在物品靠近特定区域/机关时发生（距离阈值触发）。
  - 用法：把“站点”做成可感知的磁场/泛光范围；玩家拖进去就懂“这里不一样”。
- **停留/静置（Pause）**：玩家什么都不做本身就是有效操作（停留 X 秒产生变化）。
  - 用法：发酵/融合/显现隐藏属性；配合“揭晓延迟”形成强仪式感。

#### 10.2 第二层（核心扩展动作，适合做成关卡卖点）

- **旋转/对齐（Rotate/Align）**：必须处在正确朝向/对齐状态才能生效（两指旋转或拖到对齐位）。
  - 低副作用版本：只给少量关键物（比如“杯口对齐才能倒入”“桂花朝上才释香”），避免全物品都要对齐导致疲劳。
- **踩点/激活（Trigger）**：把物品放到节点上改变环境状态（开/关、冷/热、允许某类合成）。
  - 与本 Brainstorm 的“凹槽/站点”天然一致：节点被占用 → 世界状态改变。
- **移动环境（Shift Space）**：玩家不动物品，改变世界结构（水位/昼夜/季节轮盘/站点位置）。
  - 低副作用版本：先做“昼夜切换影响可用站点/香型”或“水位升降打开/关闭区域”，不要一上来做复杂拼接地图。

#### 10.3 第三层（高级但仍可控的神秘感来源）

- **分离/提取（Extract）**：从合成物中取出成分，提供可逆性，降低试错焦虑。
  - 低副作用版本：只让少数物品可提取（你已有 `extracts`），并尽量“拖到研磨台/过滤台提取”，避免 UI 过多按钮。
- **共鸣/自动吸引（Resonance）**：匹配属性自动靠拢/排斥，减少操作负担，增强“世界是活的”。
- **互相排斥（Repulsion）**：某些物品之间存在排斥力，必须在“临界距离”完成操作，形成 Zelda 神庙式空间谜题。
  - 注意：用“抖动/推开/低频音/光冷”表现，不要弹“无法合成”文字；排斥范围要宽容，避免变成手速游戏。

### 11) “关卡目标语法”库：让每一关的目标都不一样（大量非槽位目标）

> 目标：你说的重点——**关卡目标不只是“献上/放槽里”**。  
> 做法：把目标拆成可复用的“玩法语法”，不同语法组合出不同关卡结构与节奏。

#### 11.1 唤醒沉睡的神（Hollow Knight 式“唤醒”）：目标是“让世界苏醒”

> 把“神”换成：沉睡的配方谱系 / 失温的门 / 不再歌唱的工坊 / 一段被封存的记忆。  
> 核心不是交付物品，而是完成一套**仪式链**让环境从“死”变“活”。

- **唤醒目标A：点燃/复苏（Wake）**：让场景从冷寂→有呼吸（光纹、风铃、门的心跳）。
  - 手段例：连续成功合成 3 次（世界变暖阈值）/ 完成一次发酵（时间之证）/ 完成一次净化（清澈之证）。
- **唤醒目标B：奏响（Tune）**：让背景音乐恢复旋律（不是道具奖励）。
  - 手段例：按顺序触发 3 个“音叉机关”（可由物品状态触发：冷/热/香）。
- **唤醒目标C：还原配方星座（Constellation）**：完成“谱系”的复原（桂花系/玫瑰系/茉莉系）。
  - 手段例：达成 3 次“同谱系合成”（不要求特定物品名，只要求谱系正确）。
- **唤醒目标D：净化封印（Purge）**：让污染/焦糊/混香消散，场景颜色回到温柔。
  - 手段例：做出“清水/过滤水/湿布”并使用在污染源上（环境交互点，不是槽位）。

#### 11.2 “修复世界”目标：目标是改变地图状态（而不是得到某物）

- **修桥/修路**：用“胶/糖浆/奶酪”修补裂缝，让新路径可通行（完成标准是路径点亮/角色可以通过）。
- **调水位**：把水位维持在某个区间 X 秒（目标是“稳定”，不是“合成出阀门”）。
- **熄火/控火**：把炉火从失控状态降到温柔火（目标是“火焰状态正确”）。
- **清理残香**：把花窖的“混香污染”洗掉（目标是“花窖恢复纯净”，为下一关铺垫）。

#### 11.3 “守护/维持”目标：目标是在一段时间里保持条件成立

> 这类目标能制造强张力与节奏，但不靠手速，靠规划。

- **维持冷源**：让某通道保持结冰 10 秒，同时完成一次合成/救援（撤掉冷源会塌）。
- **维持发酵温度**：发酵缸需要温度区间，玩家要在热/冷之间调节以保持稳定。
- **防止复燃**：灭火后会缓慢复燃，玩家要在复燃前完成下一步（不是计时失败惩罚，而是“节奏窗口”）。

#### 11.4 “探测/辨识”目标：目标是找到“正确的一个”而不是做出“某个”

- **真假原料**：场景里有 3 份“相似原料”，只有一份是真的（靠“靠近某区域会发光/发声”辨识）。
- **缺失线索**：目标是找到“配方纸碎片/老菜单角”，合成只是工具（例如用“灯/清水/擦拭”让字显现）。
- **味觉校准**：门给出三味条“偏甜/偏淡/偏浑”，玩家要通过调味/过滤把它校准到“刚好”（目标是达标，不是固定物名）。

#### 11.5 “选择/誓言”目标：目标是做出选择，锁定规则（换取推进）

> 这类目标特别适合“宝珠文化的仪式感”，且能自然制造变化与复玩。

- **立誓（Vow）**：选择“本关不加热 / 不混香 / 不加糖”之一，门才显形（誓言本身就是目标达成）。
- **献祭非物品**：不是献上目标物，而是献上“时间/耐心”：让某物静置到完全发酵，门才回应。
- **取舍目标**：只能携带/激活一个关键物（冷源 vs 香源），玩家必须规划顺序而不是全都堆上去。

#### 11.6 “搬运/路径”目标：目标是把某物安全送到某处（空间解谜）

- **护送易碎物**：某物经过炉火会融、经过水会散；玩家要选路线或先改变环境。
- **污染携带**：经过花窖会沾香，若沾错香门会拒绝；玩家要绕路/先净化。
- **排斥通行**：携带 A 会被某区域排斥，需要先让 C 激活/共鸣才能通过（Zelda 神庙式临界距离谜题）。

### 12) “每一关目标有什么用”：目标的意义与奖励关（不靠优惠券也想继续）

> 关键：目标不只是结算，它要承担“**教学** / **推进** / **叙事** / **呼吸** / **奖励**”的节奏功能。

#### 12.1 每关目标的 6 种用途（做关卡时先选用途）

- **用途1：教一个新动词**：解锁/首次使用一个站点或动作（靠近、静置、触发、提取、排斥…）。
- **用途2：教一个新规则**：不是教操作，而是教世界规律（热会融化、花会污染、过滤会净化、顺序会改变结果）。
- **用途3：把旧规则重混（Remix）**：两条旧规则叠加，玩家体验“我变强了”。
- **用途4：提供情绪高潮**：门/音乐/场景强回应（合成只是引线）。
- **用途5：叙事推进**：解锁记忆碎片、老菜单片段、店内一幕（短、可跳过、但有重量）。
