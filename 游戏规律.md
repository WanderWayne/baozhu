# 宝珠游戏 - 合成规律与逻辑

> 这份文档记录游戏中的合成规律，帮助设计关卡和确保玩家能通过观察学习推理。

---

## 核心规律

### 1. 发酵 + 滤布规律（鲜奶／甜奶成酪）

| 规律 | 公式 | 示例 |
|------|------|------|
| 鲜奶 + 发酵 → 酸奶 | `牛奶 + 发酵 → 酸奶` | （静置蜕变） |
| 酸奶 + 滤布 → 厚酪 | `酸奶 + 滤布 → 奶酪` | |
| 甜液体 + 发酵 → 甜酸奶 | `甜牛奶 + 发酵 → 甜酸奶` | |
| 甜酸奶 + 滤布 → 轻盈酪 | `甜酸奶 + 滤布 → 雪酪` | |

**误入**：`甜牛奶 + 酿造 → 甜奶浊酿`（轻盈酪不靠酿造）。

**与菌种线区分**：`牛奶 + 菌种 → 发酵奶 → … → 原味酸奶 / 雪域酸奶` 是另一条「菌种酸奶」支线；本篇奶酪／雪酪走「发酵 + 滤布」。

---

### 2. 鲜奶遇酿造（奶酒）

| 规律 | 公式 | 示例 |
|------|------|------|
| 鲜奶 + 酿造 → 柔奶酒 | `牛奶 + 酿造 → 奶酒` | 第二章酪饮铺垫 |

---

### 3. 调味规律（甜味添加）

| 规律 | 公式 | 示例 |
|------|------|------|
| 液体 + 冰糖碎 = 甜液体 | `液体 + 冰糖碎 → 甜液体` | 牛奶 + 冰糖碎 → 甜牛奶 |

---

### 4. 双酪规律（两种酪合并）

| 规律 | 公式 | 示例 |
|------|------|------|
| 奶酪 + 雪酪 = 双酪 | `厚酪 + 轻酪 → 双酪` | 奶酪 + 雪酪 → 双酪 |

**设计意图**：让玩家理解「双酪」是两种质地酪的组合，为后续酪饮打基础。

---

### 5. 花香酒酿规律

| 规律 | 公式 | 示例 |
|------|------|------|
| 花香物 + 酒酿原浆 = 花香酒酿 | `花 + 酒酿原浆 → 花酒酿` | 玫瑰 + 酒酿原浆 → 玫瑰酒酿 |
|  |  | 桂花 + 酒酿原浆 → 桂花酒酿 |

**推理链**：玩家在第 5 关看到「玫瑰 + 酒酿原浆 = 玫瑰酒酿」，Boss 关验证「桂花 + 酒酿原浆 = 桂花酒酿」，形成通用规律。

---

### 6. 酪饮规律

| 规律 | 公式 | 示例 |
|------|------|------|
| 双酪 + 花香酒酿 = 酒酿××酪 | `双酪 + ××酒酿 → 酒酿××酪` | 双酪 + 玫瑰酒酿 → 酒酿玫瑰酪 |
|  |  | 双酪 + 桂花酒酿 → 酒酿桂花酪 |

**设计意图**：统一的命名规则让玩家能预测产物名称。

---

### 7. 冰镇规律

| 规律 | 公式 | 示例 |
|------|------|------|
| 可冰镇物 + 冰块 = 冰×× | `xxx + 冰块 → 冰xxx` | 酒酿桂花酪 + 冰块 → 冰酒酿桂花酪 |

---

### 8. 冰酥酪（酿香甜奶路径）

| 步骤 | 公式 |
|------|------|
| 甜牛奶 + 酒酿原浆 | → 酿香奶底 |
| 酿香奶底 + 冰块 | → **霜酪** |
| 霜酪 + 冰块 **或** 霜酪 + 酒酿原浆 | → **冰酥酪** |

另：**冰酿奶 + 冰糖碎 → 冰酥酪**（备用路径）。

## 第一章关卡规律教学顺序

| 关卡 | 名称 | 教授规律 | 目标产物 |
|------|------|----------|----------|
| 101 | 第一缕甜 | 液体 + 冰糖碎 = 甜液体 | 甜牛奶 |
| 102 | 奶酿成酪 | 鲜奶发酵→酸奶；酸奶+滤布→奶酪 | 奶酪 |
| 103 | 酿酒之道 | 每种米 + 酿造 → 不同产物（四合一） | 酒酿等 |
| 104 | 双酪之约 | 甜奶发酵→甜酸奶→滤布→雪酪；再双酪合璧 | 双酪 |
| 105 | 酪饮之道 | 花 + 酒酿原浆；双酪 + 花酒酿 → 酪饮 | 酒酿玫瑰酪 |
| 106 | Boss | 综合运用以上规律 | 冰酒酿桂花酪 |

## Boss 关完整合成路径（最短示意）

**目标**：冰酒酿桂花酪  

```
初始物品：牛奶、发酵、冰糖碎、桂花、酒酿原浆、冰块、滤布

步骤1：牛奶 + 冰糖碎 → 甜牛奶
步骤2：牛奶 + 发酵 → 酸奶
步骤3：酸奶 + 滤布 → 奶酪
步骤4：甜牛奶 + 发酵 → 甜酸奶
步骤5：甜酸奶 + 滤布 → 雪酪
步骤6：奶酪 + 雪酪 → 双酪
步骤7：桂花 + 酒酿原浆 → 桂花酒酿
步骤8：双酪 + 桂花酒酿 → 酒酿桂花酪
步骤9：酒酿桂花酪 + 冰块 → 冰酒酿桂花酪
```

## 规律命名约定

为了让玩家能预测产物名称，遵循以下命名规则：

| 模式 | 命名格式 | 示例 |
|------|----------|------|
| 花香 + 酒酿原浆 | `{花名}酒酿` | 玫瑰酒酿、桂花酒酿 |
| 双酪 + 花香酒酿 | `酒酿{花名}酪` | 酒酿玫瑰酪 |
