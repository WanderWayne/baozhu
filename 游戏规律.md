# 宝珠游戏 - 合成规律与逻辑

> 这份文档记录游戏中的合成规律，帮助设计关卡和确保玩家能通过观察学习推理。

---

## 核心规律

### 1. 酿造规律（液体凝固）
| 规律 | 公式 | 示例 |
|------|------|------|
| 液体 + 酿造 = 凝固物 | `液体 + 酿造 → 酪` | 牛奶 + 酿造 → 奶酪 |
| 甜液体 + 酿造 = 轻盈酪 | `甜液体 + 酿造 → 雪酪` | 甜牛奶 + 酿造 → 雪酪 |

**推理链**：玩家先看到「牛奶 + 酿造 = 奶酪」，再看到「甜牛奶 + 酿造 = 雪酪」，可以推理出「酿造能让液体凝固」。

---

### 2. 调味规律（甜味添加）
| 规律 | 公式 | 示例 |
|------|------|------|
| 液体 + 冰糖碎 = 甜液体 | `液体 + 冰糖碎 → 甜液体` | 牛奶 + 冰糖碎 → 甜牛奶 |

---

### 3. 双酪规律（两种酪合并）
| 规律 | 公式 | 示例 |
|------|------|------|
| 奶酪 + 雪酪 = 双酪 | `厚酪 + 轻酪 → 双酪` | 奶酪 + 雪酪 → 双酪 |

**设计意图**：让玩家理解「双酪」是两种质地酪的组合，为后续酪饮打基础。

---

### 4. 花香酒酿规律
| 规律 | 公式 | 示例 |
|------|------|------|
| 花香物 + 酒酿原浆 = 花香酒酿 | `花 + 酒酿原浆 → 花酒酿` | 玫瑰 + 酒酿原浆 → 玫瑰酒酿 |
|  |  | 桂花 + 酒酿原浆 → 桂花酒酿 |

**推理链**：玩家在第4关看到「玫瑰 + 酒酿原浆 = 玫瑰酒酿」，第5关验证「桂花 + 酒酿原浆 = 桂花酒酿」，形成通用规律。

---

### 5. 酪饮规律
| 规律 | 公式 | 示例 |
|------|------|------|
| 双酪 + 花香酒酿 = 酒酿xx酪 | `双酪 + xx酒酿 → 酒酿xx酪` | 双酪 + 玫瑰酒酿 → 酒酿玫瑰酪 |
|  |  | 双酪 + 桂花酒酿 → 酒酿桂花酪 |

**设计意图**：统一的命名规则让玩家能预测产物名称。

---

### 6. 冰镇规律
| 规律 | 公式 | 示例 |
|------|------|------|
| 可冰镇物 + 冰块 = 冰xx | `xxx + 冰块 → 冰xxx` | 酒酿桂花酪 + 冰块 → 冰酒酿桂花酪 |

**推理链**：冰块让成品变成「冰」版本，是最终点睛之笔。

| 关卡 | 名称 | 教授规律 | 目标产物 |

## Boss关完整合成路径

**目标**：冰酒酿桂花酪

```
初始物品：牛奶、酿造、冰糖碎、酒酿原浆、桂花、冰块

步骤1：牛奶 + 冰糖碎 → 甜牛奶        （调味规律）
步骤2：牛奶 + 酿造 → 奶酪            （酿造规律）
步骤3：甜牛奶 + 酿造 → 雪酪          （酿造规律-轻盈版）
步骤4：奶酪 + 雪酪 → 双酪            （双酪规律）
步骤5：桂花 + 酒酿原浆 → 桂花酒酿    （花香酒酿规律）
步骤6：双酪 + 桂花酒酿 → 酒酿桂花酪  （酪饮规律）
步骤7：酒酿桂花酪 + 冰块 → 冰酒酿桂花酪  （冰镇规律）
```

---

## 规律命名约定

为了让玩家能预测产物名称，遵循以下命名规则：

| 模式 | 命名格式 | 示例 |
|------|----------|------|
| 花香 + 酒酿原浆 | `{花名}酒酿` | 玫瑰酒酿、桂花酒酿 |
| 双酪 + 花酒酿 | `酒酿{花名}酪` | 酒酿玫瑰酪、酒酿桂花酪 |
| 成品 + 冰块 | `冰{成品名}` | 冰酒酿桂花酪 |

---

## 干扰项设计原则

在「香气之约」等关卡中加入干扰项时：

1. **干扰项不能与正确路径混淆**：如给了牛奶和冰糖碎，但目标不需要甜牛奶
2. **干扰项要让玩家验证规律**：如玩家可能尝试「牛奶 + 桂花」，发现无效
3. **正确路径唯一**：确保只有一条路能到达目标
